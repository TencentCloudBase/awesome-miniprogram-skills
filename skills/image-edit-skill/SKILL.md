# AI 图片编辑

## 触发场景

用户原话举例：

- **风格转换**："把这张图改成水墨画风格"、"变成动漫风格"
- **背景替换**："把背景换成白色"、"把室内背景换成海边"
- **内容修改**："把猫换成狗"、"给画面加一些樱花飘落"
- **扩图生成**："把画面往右延伸"

## 不适用范围

- 从零生成的图片（无原图）→ 不在本技能范围，由 image-gen-skill 处理
- 文本生成需求 → 不在本技能范围，由 text-gen-skill 处理
- 需要精确到像素级别的专业修图 → 不在本技能范围，AI 无法保证像素级精度

## 接口链路

- `editImage`：基于原图和编辑需求，通过 AI 重绘实现图片编辑效果

## 使用顺序

1. 用户需要提供一张原图（图片 URL）和编辑描述
2. 基于编辑描述生成新的 prompt，调用图片生成模型重新生成
3. 返回原图和编辑后图片的对比
4. 编辑后的图片自动持久化到云存储

## 设计约束

- 此为 AI 重绘实现的编辑，非像素级精确编辑，不能保证与原图完全一致
- 必须通过云函数调用，小程序端不直接处理图片
- 原图需要提供可访问的网络 URL
