# water-tracker

喝水记录，支持记录每日饮水量、查看饮水历史和今日进度。

## 功能

- 记录本次喝水量，查看当天累计和目标进度
- 查看最近日期的饮水记录

## 用户输入示例

- "我刚喝了 250 毫升水，帮我记一下"
- "记录一杯 300ml 的水"
- "今天喝了多少水了"
- "看下我最近几天的喝水情况"
- "帮我回顾一下今天的饮水记录"

## 原子接口

| 接口名 | 说明 |
|--------|------|
| `addWaterRecord` | 记录一次喝水量 |
| `getWaterRecords` | 查询饮水记录 |

## 原子组件

| 组件路径 | 说明 |
|---------|------|
| `components/add-water-result/index` | 喝水记录结果卡片 |
| `components/water-records/index` | 饮水记录列表卡片 |

## 后端依赖

| 资源 | 名称 |
|------|------|
| 云函数 | `water-tracker-handler` |
