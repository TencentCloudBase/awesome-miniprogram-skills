# 简单待办清单

基于云开发数据库直接完成待办事项查询、添加、完成切换与删除的简单教学示例。

## 触发场景
用户原话举例（路由命中本技能）：
- "帮我看下待办列表"
- "新增一个待办：明天交周报"
- "把『明天交周报』标记完成"
- "删掉那个买咖啡豆的待办"
- "我的 todo 还有哪些没做"

## 不适用范围
- 复杂项目管理、多人协作、日历提醒、文件附件等诉求 → 不在本技能范围
- 门店排队、下单支付、导航等诉求 → 不在本技能范围

## 使用顺序
- 查看待办列表无需前置条件，可直接查询。
- 修改某条待办状态前需先拿到真实 todoId；没有 todoId 时先展示列表，再从列表中选择具体待办。
- 新增、切换完成状态、删除后都重新展示最新待办列表。
